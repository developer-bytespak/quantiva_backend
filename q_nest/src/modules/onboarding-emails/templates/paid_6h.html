To connect Quantiva to your exchange, you'll need an API key. Here's how:

  1. Sign in to your exchange (Binance / Bybit / Alpaca).
  2. Go to Account → API Management and create a new API key.
  3. Enable spot trading permissions only (no withdrawals — we never need them).
  4. Copy the API key and secret into Quantiva.
